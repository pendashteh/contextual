This document is meant for developers of this project, or those who seek to extend it.

## Features
- [x] Work with TMUX.
- [ ] Switch to the latest working directory (LWD)
- [x] Run the default command on start.


### Run default command
On the start of a session, the content of file `command` under the `CONTEXT_PATH` gets executed in bash.
